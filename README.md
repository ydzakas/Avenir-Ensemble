# Avenir Ensemble - Soutenir les Étudiants Togolais

## Description

Ce projet vise à soutenir les étudiants togolais ayant perdu un parent en leur fournissant un soutien éducatif, psychologique et communautaire. Le site web présente notre mission, notre impact et offre des moyens de nous contacter pour plus d'informations ou pour offrir de l'aide.

## Contenu

- `index.html` : Contient la structure principale de la page web, y compris les sections d'accueil, de mission, d'impact et de contact.
- `style.css` : Contient les styles CSS pour la page web, y compris les styles personnalisés et ceux fournis par Tailwind CSS.

## Comment utiliser

1. **Cloner ce dépôt** :
   ```sh
   git clone https://github.com/ton-utilisateur/ton-depot.git
   ```

2. *Naviguer vers le répertoire du projet* :
   ```sh
   cd ton-depot
   ```

3. *Ouvrir le fichier `index.html` dans un navigateur web* pour voir le site en action.

*Technologies utilisées*
- *HTML5* : Pour la structure du site web.
- *Tailwind CSS* : Pour les styles modernes et réactifs.
- *Google Fonts* : Pour une typographie élégante.
- *AOS Animation Library* : Pour des animations douces et engageantes.
- *CSS personnalisé* : Pour des effets de carte et des styles supplémentaires.
